Mapping

What is mapping?

Mapping is the process of defining how a document, and the fields it contains, are stored and indexed.

The pizza mapping

Define the data types of the document fields


{
  "pizza": {
    "properties": {
      "id": {
        "type": "long"
      },
      "name": {
        "type": "string"
      },
      "topping": {
        "type": "string",
        "index": "not_analyzed"
      },
      "weight": {
        "type": "long"
      }
    }
  }
}

Query and filter context

Query context Filter context
Full text search Excat match
Relevance scoring Binary yes/no
Relativly slow Fast
Not cacheable Cacheable

Term-based vs full-text

Term-based Full-text
  • No analysis phase
  • Operates on a single term
  • Requires exact match
  • Understands field mapping
  • Query is passed though an analyzer
  • Analyzer produces list of terms to be queried

The match query

  • The first query you should consider
  • High-level full-text query

Example:


"query": {
    "match": {
        "title": "FOO!"
    }
}

Resulting search term: 'foo'
